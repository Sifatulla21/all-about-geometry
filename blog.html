Some Important Questions

1.Dom:DOM stands for Document Object Model. It handel various types of events of a button. By using DOM we can change the stracture of a page. There are many use of DOM also.

2.Purpose of DOM:

1. It helps us to manipulate the content and structure of a web page dynamically

2. We can make style also by using DOM

Select HTML elements using DOM: There are many methods to select HTML elements using DOM. They are

1.getElementById()

2.getElementsByClassName()

3.getElementsByTagName

4.querySelector()

5.querySelectorAll()

Event bubble: Event bubble is a DOM behavior where if we perform an action on a HTML element it will also occurs on all the parents element. This is happen because the event bubble from the inner most element to the outer most element.

Callback function: Call back function is a function that is pass as an argument of another function.

We use call back function because it can improve the performance and responsiveness of our system
